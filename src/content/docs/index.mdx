	<Card title="Portabilidad y Versatilidad" icon="laptop">
		Java es conocido por su capacidad de ser ejecutado en múltiples plataformas sin necesidad de recompilación. Esto significa que puedes escribir una vez y ejecutar en cualquier lugar, lo que simplifica enormemente el desarrollo y la distribución de aplicaciones.
	</Card>
	<Card title="Ecosistema Robusto" icon="seti:lock">
		Java cuenta con un vasto ecosistema de bibliotecas, frameworks y herramientas que facilitan el desarrollo de aplicaciones de cualquier tamaño y complejidad. Desde frameworks de desarrollo web como Spring hasta herramientas de construcción como Maven, Java ofrece todo lo necesario para construir aplicaciones sólidas y escalables.
	</Card>
	<Card title="Seguridad y Fiabilidad" icon="setting">
		Java se ha ganado una reputación por su enfoque en la seguridad y la fiabilidad. Con su arquitectura de sandbox y su sistema de gestión de memoria robusto, Java proporciona un entorno seguro y estable para el desarrollo de aplicaciones críticas para el negocio.
	</Card>
	<Card title="Comunidad Activa y Soporte Continuo" icon="open-book">
		Java cuenta con una comunidad global de desarrolladores apasionados y expertos que están constantemente trabajando en mejorar el lenguaje y su ecosistema. Además, Oracle, la empresa detrás de Java, proporciona soporte técnico continuo y actualizaciones regulares para garantizar que Java siga siendo relevante y eficiente en el futuro.
