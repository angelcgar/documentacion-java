## ¿Qué encontrarás aquí?

- **Fundamentos sólidos**: Sintaxis, tipos de datos, POO, excepciones y colecciones explicados claramente.
- **JVM en profundidad**: Entiende cómo funciona realmente Java bajo el capó.
- **Herramientas modernas**: Maven, Gradle, IDEs y gestión de dependencias.
- **Buenas prácticas**: Patrones, convenciones y errores comunes a evitar.
- **Testing real**: JUnit, Mockito y estrategias de testing efectivas.
- **Backend moderno**: Spring Boot, APIs REST y arquitectura escalable.
- **Kotlin Multiplatform**: Una introducción opcional para desarrolladores Java interesados en KMP.

Esta documentación se construye con experiencia real en proyectos Java. No es un simple proyecto de portafolio, sino un recurso que busca aportar valor real a la comunidad.
